# Frontend

## Опис

Фронтенд для проєкту учнівського самоврядування **ЕРА** - міністерства культури та мистецтва Березанського ліцею №3. Ідея проста: зробити зручний хаб для учнів та вчителів, де можна буде

- отримати розклад уроків*
- дізнатись найближчі шкільні події

\* - знаходиться в розробці

> Проект тільки почав рости, але буде потужно :P

## Технології

- React
- TypeScript
- Axios

## Функціонал 

- **Авторизації в адмін-панель**
    Працює через backend, логін/пароль зберігаються в PostgreSQL.

- **Роутінг**
    Нормальна навігація між сторінками.

- **HTTP запити до бекенду**

## Установка та запуск

> Інструкція передбачає встановлений npm.

```bash
npm install
npm start
```

Фронт піднімається до localhost, бек треба запустити окремо.

## Backend

Фронтенд працює разом із backend-сервесом (Spring + PostgreSQL)

На беку вже реалізовано:

- авторизація через JWT
- CRUD для подій ліцею
- зберігання адмінів у PostgreSQL
- декілька ендпоінтів
- налаштовані конфіги безпеки

Поки що там базові речі, бо проект на стадії зародження, але розганяється.

## Деплой

наразі проект працює **локально**.

Після релізу планується деплой або в школі або на будь-який хостинг.
